# Portfolio Frontend (Next.js)

This is the Next.js frontend component of Ethan Nien's developer portfolio, styled with a custom vanilla CSS design system.

## Getting Started

To run the frontend, you must execute the commands from the **`frontend`** directory.

If you are at the repository root, first navigate to the frontend folder:
```bash
cd frontend
```

Then, install dependencies (if not already done):
```bash
npm install
```

Now, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser to view the application.

---

## 🎨 Design Theme & Styles

All styles are configured via CSS variables inside `src/app/globals.css`. You can easily swap themes by adjusting variable values for backgrounds, cards, buttons (olive green), and highlights (rust orange).
